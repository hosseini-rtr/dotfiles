# 🛠️ Dotfiles

Personal configuration files for my development environment.

This repository contains the configuration files (dotfiles) I use across my systems — including shell, editor, git, and other development tools.

> Keep your environment reproducible. Keep it versioned.


---
## 🧠 Philosophy

* Minimal but powerful
* Fast shell startup
* Clean and readable configs
* Version controlled
* Portable across machines

---

## 💻 Supported Systems

* macOS
* Linux (Debian/Ubuntu based)

---

## 📦 Example Structure

```
dotfiles/
├── zsh/
│   ├── .zshrc
│   └── README.md (optional)
├── git/
│   ├── .gitconfig
│   └── README.md (optional)
├── nvim/
├── tmux/
└── README.md
```

## Notes

* If fork, No secrets are stored here.
* Machine-specific values should not be committed.
* Designed to keep my setup reproducible across systems.
